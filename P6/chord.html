Interpreting the diagram

Chord diagrams present information using two sets elements: (1) a set of arcs around the outside of the diagram and (2) a set of chords or ribbons on the inside.

The arc length shows the aggregated averaged delay. The chords show relations between arcs (airports). Their width on either end is determined by the average delay between airpoirts. The color represent the direction in which delay is higher.
